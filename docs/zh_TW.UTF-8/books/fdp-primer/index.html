摘要

感謝您參與 FreeBSD 文件計劃(簡稱：FDP, FreeBSD Documentation Project)，您的點滴貢獻，都相當寶貴。

本入手書內容包括：如何開始著手翻譯的各項細節，以及會用到的一些好用工具(包括：必備工具、輔助工具) ，以及文件計畫的宗旨。

本文件還在草稿，尚未完稿。未完成的章節，我們會在章節名稱旁邊加註『* 』以作識別。
