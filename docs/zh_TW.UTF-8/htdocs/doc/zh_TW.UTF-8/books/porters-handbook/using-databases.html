6.19. Using databases

表格 6.40. Variables for ports using databases
VariableMeans
USE_BDBIf variable is set to yes, add dependency on databases/db41 port. The variable may also be set to values: 2, 3, 40, 41, 42, 43, 44, 45 46, or 47. You can declare a range of acceptable values, USE_BDB=42+ will find the highest installed version, and fall back to 42 if nothing else is installed.
USE_MYSQLIf variable is set to yes, add dependency on databases/mysql50-server port. An associated variable, WANT_MYSQL_VER, may be set to values such as 323, 40, 41, 50, 51 or 60.
USE_PGSQLIf set to yes, add dependency on databases/postgresql82 port. An associated variable, WANT_PGSQL_VER, may be set to values such as 73, 74, 80, 81, 82, or 83.
