10. Outros

Como root você pode desmontar o CDROM com /sbin/umount /cdrom, retirá-lo do drive, inserir outro, e montá-lo com/sbin/mount_cd9660 /dev/cd0a /cdrom,assumindo que cd0a é o nome do seu dispositivo de CDROM. As versões mais recentes do FreeBSD deixam você montar o CDROM apenas com /sbin/mount /cdrom.

Usar o sistema de arquivos live — o segundo dos discos de CDROM do FreeBSD — é útil se você tem um espaço limitado. O conteúdo do sistema de arquivos live varia de versão para versão. Você pode tentar jogar a partir do CDROM. Isso envolve o uso do lndir, que é instalado com o X Window System, para informar aos programas onde encontrar os arquivos necessários, pois eles estão no sistema de arquivos de /cdrom, em vez de /usr e seus subdiretórios, que é o local onde eles esperam estar. Consulte man lndir.
