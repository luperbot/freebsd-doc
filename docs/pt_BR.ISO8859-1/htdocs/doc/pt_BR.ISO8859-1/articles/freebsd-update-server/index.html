Resumo

Este artigo descreve como construir um Servidor de Atualização do FreeBSD para uso interno na sua organização. O software freebsd-update-server foi escrito pelo Colin Percival, Chefe de Segurança emérito do FreeBSD. Para usuários que acreditam que é conveniente atualizar seus sistemas a partir de um servidor oficial de atualização, construir o seu próprio Servidor de Atualização do FreeBSD pode ajudá-lo a estender suas funcionalidades, seja por adicionar suporte a versões customizadas do FreeBSD ou por viabilizar a criação de um servidor local o qual permitirá atualizações mais rápidas caso você possua muitos servidores para atualizar.
