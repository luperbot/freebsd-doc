    Einführung

    Hier findet sich eine Liste an Plattformen, die FreeBSD momentan unterstützt, sowie Plattformen, auf die gerade portiert wird.

    Inhaltsverzeichnis

    • FreeBSD/amd64 Project
    • FreeBSD/ARM Project
    • FreeBSD/i386 Project
    • FreeBSD/ia64 Project
    • FreeBSD/MIPS Project
    • FreeBSD/pc98 Project
    • FreeBSD/ppc Project
    • FreeBSD/sparc64 Project
    • FreeBSD/xbox Project

    Kommentare und Feedback

    Haben Sie Anmerkungen zu einer Portierung, oder wünschen Sie den Entwicklern Rückmeldungen zu geben, schicken Sie diese an die jeweilige Mailingliste. Die Listen sind:

    • Für die FreeBSD/amd64-Portierung, freebsd-amd64@FreeBSD.org
    • Für die FreeBSD/ARM-Portierung, freebsd-arm@FreeBSD.org
    • Für die FreeBSD/i386-Plattform, freebsd-current@FreeBSD.org
    • Für die FreeBSD/ia64-Portierung, freebsd-ia64@FreeBSD.org
    • Für die FreeBSD/MIPS-Portierung, freebsd-mips@FreeBSD.org
    • Für die FreeBSD/pc98-Portierung, re-pc98@FreeBSD.org
    • Für die FreeBSD/ppc-Portierung, freebsd-ppc@FreeBSD.org
    • Für die FreeBSD/sparc64-Portierung, freebsd-sparc@FreeBSD.org
    • Für die FreeBSD/xbox-Portierung, freebsd-current@FreeBSD.org
    • Für allgemeine Architekturfragen, freebsd-arch@FreeBSD.org
