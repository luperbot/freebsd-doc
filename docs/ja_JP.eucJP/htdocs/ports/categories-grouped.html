    FreeBSD Ports -- グループによる分類

    ports の検索:

    主にエンドユーザ向けの ports

    • Accessibility (61) -- 障害を持ったユーザの役に立つ ports。
    • Audio (902) -- オーディオユーティリティ - ほとんどにサポートするサウンドカードが必要。
    • Deskutils (306) -- さまざまなデスクトップユーティリティ。
    • Editors (325) -- 一般的なテキストエディタ。
    • Finance (249) -- 金融や財務会計に関連するアプリケーション。
    • Games (1233) -- 種々雑多なゲーム。
    • Graphics (1180) -- グラフィックライブラリとユーティリティ。
    • Multimedia (468) -- マルチメディアソフトウェア。
    • Net-p2p (108) -- Peer to peer ネットワークのためのソフトウェア。
    • Print (326) -- 印刷処理のユーティリティ。
    • Shells (46) -- 各種シェル (tcsh, bash など)。
    • Textproc (1805) -- テキスト処理/検索ユーティリティ。

    主にシステム管理者および開発者向けの ports

    • Archivers (236) -- アーカイビングとアンアーカイビングのユーティリティ。
    • Databases (988) -- データベースソフトウェア。
    • Devel (5694) -- ソフトウエア開発ユーティリティ及びライブラリ。
    • Emulators (190) -- 他の OS をエミュレートするユーティリティ。
    • Java (530) -- Java 言語サポート。
    • Lang (471) -- コンピュータ言語。
    • Ports-mgmt (70) -- ports および packages 管理のためのユーティリティ。
    • Security (1205) -- システムセキュリティソフトウエア。
    • Sysutils (1185) -- 各種システムユーティリティ。

    コンピュータ通信およびインターネットに関連した ports

    • Comms (205) -- 通信ユーティリティ。
    • Dns (198) -- DNS クライアントとサーバユーティリティ。
    • Ftp (113) -- FTP クライアントとサーバユーティリティ。
    • Irc (140) -- インターネットリレーチャットのユーティリティ。
    • Mail (774) -- 電子メールパッケージとユーティリティ。
    • Net (1560) -- ネットワーキングユーティリティ。
    • Net-im (196) -- インスタントメッセージングのためのソフトウェア。
    • Net-mgmt (343) -- ネットワーク管理ユーティリティ。
    • News (91) -- USENETニュースサポートソフトウェア。
    • Www (2999) -- WEB ユーティリティ (ブラウザ、HTTP サーバ など)。

    X Window System に関連した ports

    • X11 (515) -- X Window System ベースのユーティリティ。
    • X11-clocks (55) -- X Window System ベースの時計。
    • X11-drivers (48) -- X Window System のドライバ。
    • X11-fm (37) -- X Window System ベースのファイルマネージャ。
    • X11-fonts (240) -- X Window System のフォントとフォントユーティリティ。
    • X11-servers (12) -- X Window System サーバ。
    • X11-themes (157) -- X Window System のテーマ。
    • X11-toolkits (278) -- X Window System ベースの開発ツールキット。
    • X11-wm (138) -- X Window System のウィンドウマネージャ。

    科学や工学に関連した ports

    • Astro (131) -- 天文学関係のアプリケーション。
    • Biology (94) -- 生物学に関連するソフトウェア。
    • Cad (103) -- CAD (Computer Aided Design) ユーティリティ。
    • Math (721) -- 数値計算ソフトウエア。
    • Science (206) -- 科学に関連するソフトウェア。
