奥付

この本は 「The FreeBSD Documentation Project」 に対するたくさんの人々の共同作業によって完成したものです。 文書は DocBook DTD を用いた XML 形式で管理されており、 XSLT を使って XML からさまざまな可読形式に組版されています。 もし、Donald Knuth 氏による組版言語 TeX、 Leslie Lamport 氏による LaTeX および Sebastian Rahtz 氏による JadeTeX マクロパッケージがなかったら、 この文書の書籍版を実現することはできなかったでしょう。
