20.?Issues Specific to Developers Who Are Not Committers
--------------------------------------------------------

.. raw:: html

   </div>

.. raw:: html

   </div>

.. raw:: html

   </div>

A few people who have access to the FreeBSD machines do not have commit
bits. Almost all of this document will apply to these developers as well
(except things specific to commits and the mailing list memberships that
go with them). In particular, we recommend that you read:

.. raw:: html

   <div class="itemizedlist">

-  `Administrative Details <index.html#admin>`__

-  `Conventions <conventions.html#conventions-everyone>`__

   .. raw:: html

      <div class="note" xmlns="">

   Note:
   ~~~~~

   You should get your mentor to add you to the “Additional
   Contributors”
   (``doc/en_US.ISO8859-1/articles/contributors/contrib.additional.xml``),
   if you are not already listed there.

   .. raw:: html

      </div>

-  `Developer Relations <developer.relations.html>`__

-  `SSH Quick-Start Guide <ssh.guide.html>`__

-  `The FreeBSD Committers' Big List of Rules <rules.html>`__
