7.1.?Subversion (SVN)
---------------------

.. raw:: html

   </div>

.. raw:: html

   </div>

.. raw:: html

   </div>

Subversion (“SVN”) is a system to handle multiple versions of text files
and tracking who committed what changes and why. A project lives within
a “repository” and different versions are considered different
“branches”.
