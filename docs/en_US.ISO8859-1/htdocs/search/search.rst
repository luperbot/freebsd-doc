Mailing list archives
---------------------

The `mailing list archive <http://www.FreeBSD.org/mail/>`__ indexes are
now updated weekly!

An alternative way to read the mailing lists archives is to use the
`Mailman/Pipermail list
archive <http://lists.FreeBSD.org/mailman/listinfo>`__ (note that this
archive only carries messages from March 2003 onward).

The mailing lists (as well as many others) have also been archived by
`MarkMail <http://markmail.org/browse/?q=list:org.freebsd>`__.
