Web Resources
=============

In the real world...
--------------------

`FreeBSD in the Press <../news/press.html>`__
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Articles in the press about FreeBSD.

Newsgroups
----------

The following newsgroups contain discussion pertinent to FreeBSD users:

-  `comp.unix.bsd.freebsd.announce <news:comp.unix.bsd.freebsd.announce>`__
   (moderated)

-  `comp.unix.bsd.freebsd.misc <news:comp.unix.bsd.freebsd.misc>`__

-  `comp.unix.bsd.misc <news:comp.unix.bsd.misc>`__

Additional resources
--------------------

`The Source Code <http://fxr.watson.org/>`__
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

If you like digging your fingers into source code, here is a hypertext
version of the FreeBSD *kernel* source. This is brought to you courtesy
of Robert Watson.

`The FreeBSD Diary <http://www.freebsddiary.org/>`__
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The FreeBSD Diary is a collection of how-to entries aimed at UNIX
novices. The aim is to provide a set of step-by-step guides to
installing and configuring various ports.

`The O'Reilly Network BSD Dev Center <http://www.OnLamp.com/bsd/>`__
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The BSD Dev Center includes two regular columns on FreeBSD, featuring
tips and tutorials.

`The FreeBSD Forums <https://forums.FreeBSD.org>`__
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The Official FreeBSD Forums, offering a forum dedicated to FreeBSD and
FreeBSD aficionados.

`BSD Guides <http://www.bsdguides.org>`__
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

BSDGuides.org hosts many BSD-related tutorials.
