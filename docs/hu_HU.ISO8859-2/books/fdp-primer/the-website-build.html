8.2. A honlapok elõállítása

Miután az elõbb tárgyalt módszerek valamelyikével elõkészítettük rendszerünkön a honlapok forrásainak egy naprakész másolatát, készen állunk a honlapok létrehozására. A példánkban az ehhez használt munkakönyvtár a /usr/build volt, ahol már minden szükséges állomány megtalálható.

  1. Lépjünk be a munkakönyvtárba:

    # cd /usr/build

  2. A honlapok elõállítása a www/en könyvtárból indul, az all make(1) cél végrehajtásával megkezdõdik a honlapok készítése.

    # cd www/en
# make all
