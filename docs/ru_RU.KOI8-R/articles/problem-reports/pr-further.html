7. Дополнительная литература

Это список информационных ресурсов, относящихся к правильному написанию и обработке сообщений о проблемах. Он, без сомнения, не полон.

  • How to Report Bugs Effectively-прекрасное эссе, которое написал Simon G. Tatham о составлении полезных (не специфичных для FreeBSD) сообщений о проблемах.

  • Problem Report Handling Guidelines-интересный взгляд на обработку сообщений о проблемах самими разработчиками FreeBSD.
