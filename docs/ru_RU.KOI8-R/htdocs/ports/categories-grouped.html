    Категории Портов FreeBSD по группам

    Поиск портов:

    Порты, предназначенные для обычных пользователей

    • Accessibility (61) -- Порты в помощь людям с ограниченными возможностями.
    • Audio (902) -- Утилиты для работы со звуком - большинству требуется звуковой адаптер.
    • Deskutils (306) -- Различные утилиты для рабочего стола.
    • Finance (249) -- Приложения, связанные с деньгами и финансами
    • Games (1233) -- Разнообразнейшие игрушки.
    • Graphics (1180) -- Утилиты и библиотеки для работы с графикой.
    • Multimedia (468) -- Программное обеспечения для работы с мультимедиа.
    • Net-p2p (108) -- Приложения для работы с пиринговыми (P2P) сетями.
    • Print (326) -- Инструменты для работы с печатью.
    • Shells (46) -- Различные оболочки (tcsh, bash, и т.д.).
    • Textproc (1805) -- Утилиты для текстовой обработки/поиска.

    Порты, предназначенные для системных администраторов и разработчиков

    • Archivers (236) -- Утилиты для архивирования и разархивирования данных.
    • Databases (988) -- Программное обеспечение для баз данных.
    • Devel (5694) -- Утилиты и библиотеки для разработки программного обеспечения.
    • Editors (325) -- Обычные текстовые редакторы.
    • Emulators (190) -- Программы эмуляции других типов ОС.
    • Java (530) -- Поддержка языка Java.
    • Lang (471) -- Языки программирования.
    • Ports-mgmt (70) -- Утилиты для управления портами и пакетами.
    • Security (1205) -- Программы, обеспечивающие безопасность системы.
    • Sysutils (1185) -- Различные системные утилиты.

    Порты для компьютерных коммуникаций и для работы с Интернет

    • Comms (205) -- Коммуникационные утилиты.
    • Dns (198) -- Клиентские и серверные утилиты DNS.
    • Ftp (113) -- Клиенты и серверы FTP.
    • Irc (140) -- Утилиты для Internet Relay Chat.
    • Mail (774) -- Утилиты и пакеты для работы с электронной почтой.
    • Net (1560) -- Утилиты для работы с сетью.
    • Net-im (196) -- Приложения для мгновенного обмена сообщениями.
    • Net-mgmt (343) -- Утилиты управления сетью.
    • News (91) -- Поддержка телеконференций USENET.
    • Www (2999) -- Утилиты для WEB (браузеры, серверы HTTP, и т.д.).

    Порты X Window System

    • X11 (515) -- Утилиты для X Window System.
    • X11-clocks (55) -- Часы для X Window System.
    • X11-drivers (48) -- Драйвера для X Window System.
    • X11-fm (37) -- Менеджеры файлов для X Window System.
    • X11-fonts (240) -- Шрифты для X Window System и утилиты для работы с ними.
    • X11-servers (12) -- Серверы для X Window System.
    • X11-themes (157) -- Темы для X Window System.
    • X11-toolkits (278) -- Пакеты разработки приложения для X Window System.
    • X11-wm (138) -- Оконные менеджеры для X Window System.
