Zusammenfassung

Dies ist die FAQ für die FreeBSD-Versionen 7.X 8.X und 9.X. Alle Einträge sollten für FreeBSD ab Version 7.X relevant sein, andernfalls wird darauf explizit hingewiesen. Falls Sie daran interessiert sein sollten, an diesem Projekt mitzuarbeiten, senden Sie eine Mail an die Mailingliste 'FreeBSD German Documentation Project' <de-bsd-translators@de.FreeBSD.org>. Die aktuelle Version dieses Dokuments ist ständig auf dem FreeBSD World-Wide-Web-Server verfügbar. Sie kann aber auch als eine einzige große HTML-Datei, als Textdatei, als PostScript®- oder PDF-Datei sowie in verschiedenen anderen Formaten vom FreeBSD FTP-Server heruntergeladen werden. Alternativ können Sie die FAQ auch durchsuchen.
