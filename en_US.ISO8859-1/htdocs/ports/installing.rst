Installing the FreeBSD Ports Collection
=======================================

Search ports for: All Package Name Description Long description
Maintainer Requires

--------------

The FreeBSD installer programs allow the Ports Collection to be
installed at the same time as the operating system. See
```bsdinstall`` <../cgi/man.cgi?query=bsdinstall>`__ for FreeBSD?9.X and
later versions, or ```sysinstall`` <../cgi/man.cgi?query=sysinstall>`__
for FreeBSD?8.X and earlier versions.

If the Ports Collection is not initially installed, it can be added with
```portsnap`` <../doc/en_US.ISO8859-1/books/handbook/ports-using.html>`__
or `Subversion <../doc/en_US.ISO8859-1/books/handbook/svn.html>`__.

A `gzipped tarball of the Ports
Collection <ftp://ftp.FreeBSD.org/pub/FreeBSD/ports/ports/ports.tar.gz>`__
can also be downloaded. The current size of the file is about 56
megabytes.
