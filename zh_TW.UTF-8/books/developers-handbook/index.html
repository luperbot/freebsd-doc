摘要

歡迎使用 Developers' Handbook！ 這份文件是由許多人 不斷撰寫 而成的， 而且許多章節仍需更新或者內容還是一片空白， 如果你想幫忙 FreeBSD 文件計劃， 請寄信到 FreeBSD documentation project 郵遞論壇。

最新版的文件都在 FreeBSD 官網 上面， 也可從 FreeBSD FTP server 下載不同格式的資料。 當然也可以在其他的 mirror站下載。
