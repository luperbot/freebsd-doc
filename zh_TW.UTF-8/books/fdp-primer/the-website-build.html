8.2. Build the web pages from scratch

  1. 先建立要編譯的目錄(至少要有 60MB 空間)，並切換到該目錄。

    # mkdir /var/tmp/webbuild
# cd /var/tmp/webbuild

  2. 從 CVS tree 內 checkout 相關的 SGML 檔。

    # cvs -R co www doc

  3. 切到 www/en 目錄，然後打 make(1) all 來產生網頁。

    # cd en
# make all
